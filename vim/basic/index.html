Vim tip(Your map list in use)

Neste poste vem uma dica rápida sobre Vim.
As vezes podemos esquecer o que mapeamos no nosso querido editor. Procurando por respostas, deparei-me comuma resposta no stackoverflow sobre isso. Para descobrir é simples, basta executar o comando no Vim ;

  • Para modo normal 
     :nmap
  • Para modo visual 
     :vmap
  • Para modo inserção 
     :imap

    ;D. Simples!
